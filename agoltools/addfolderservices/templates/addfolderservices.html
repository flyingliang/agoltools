This tool will add all services from an ArcGIS Server folder into your ArcGIS Online account.

All services will be added to a new folder in your online account. The folder will be named [ServicesFolderName] - [RandomString].

For instance, if you specify the Services Folder URL as http://services.arcgisonline.com/ArcGIS/rest/services/Elevation, the folder in your online account will be named Elevation - [RandomString].

The name is suffixed with a random string to avoid name clashes with an existing folder of a similar name.
